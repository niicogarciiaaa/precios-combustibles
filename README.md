# Precios Combustibles

Aplicación Angular para consultar precios actualizados de combustibles desde el MITECO.

## Características

- Consulta en tiempo real de precios de combustibles
- Datos actualizados desde la API del MITECO
- Componentes reutilizables
- Búsqueda y filtrado de gasolineras
- Actualización automática de precios

## Instalación

```bash
npm install
```

## Ejecutar aplicación

```bash
npm start
```

La aplicación estará disponible en `http://localhost:4200/`

## API del MITECO

Esta aplicación utiliza la API pública del MITECO (Ministerio para la Transición Ecológica):
- URL: https://sedeaplicaciones.minetur.gob.es/ServiciosRESTCarburantes/PreciosCarburantes/EstacionesTerrestres/

Los datos se actualizan periódicamente desde la fuente oficial.
